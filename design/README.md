# My Messenger Design Workspace

This folder is reserved for visual design work and concept exploration.

Production Flutter code lives in:

- `apps/messenger_app/lib/main.dart`
- `apps/messenger_app/pubspec.yaml`
- `apps/messenger_app/android/app/build.gradle.kts`

Do not edit production app, server, deployment, or secret files while preparing design concepts. Use this folder for mockups, style guides, tokens, animation notes, and handoff documents.

Expected designer outputs:

- `design/concepts/concept-1.html`
- `design/concepts/concept-2.html`
- `design/concepts/concept-3.html`
- `design/concepts/README.md`
- `design/status.md`
